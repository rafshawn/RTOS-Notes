# DAC: Discretionary Access Control
- Allows file owners to control access, implemented through Unix permissions and ACLs.

# Querying User, Group ID:
- Functions like and retrieve the current user's ID or group ID.

# Changing File Owner:
- File ownership can be set using command,
- but permission issues may arise.

# Superuser:
- Also known as root user with UID 0
- has extensive privileges to change file ownership, permissions, mount filesystems, and configure devices.

# Root-only Programs:
- Many tools require root privileges for tasks like filesystem management, package management, and service control.

# ACLs: Access Control Lists
- Used for complex authorization on multi-user shared systems,
- with an explicit list of users and groups having independent permissions.
